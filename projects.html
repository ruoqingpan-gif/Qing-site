Room

一款基于 Godot 4.4 的文字推理小游戏

About This Project

《Room》是一款基于Godot 4.4引擎开发的文字推理小游戏，玩家通过点击场景内的道具触发对话，从中收集关键线索，并依据对话内容推断解答关于房间主人的相关谜题。本版本为学习开发过程中的阶段性成果记录，仅呈现了游戏玩法的一个片段。

Architecture

架构 Architecture

Scene 游戏场景
Room Board GuessScene
UI State 界面状态
StartMenuScene SettingMenuScene PauseScene
Component 组件层
Interactable
DialogItem（对话触发） Teleporter（场景传送）
System 系统层
SceneChanger Dialogic（对话插件）

逻辑流 Logic Flow

StartMenuScene 主菜单
开始游戏 → Room
Room / Board 线索收集场景
点击物体 → Interactable → Dialogic 对话
场景切换 → Teleporter → SceneChanger
暂停 / 设置 → PauseScene / SettingMenuScene
GuessScene 线索推理场景
Dialogic 时间线 → 回答问题
答错 → 重新回答  |  答对 → 下一题
